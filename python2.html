.Python Snake Game project.

The Python Snake Game, implemented with the Turtle graphics library, is a classic arcade-style game where players control a snake through arrow key inputs. The objective is to eat blue food squares, increasing the snake's length while avoiding collisions with the screen edges and the snake's body. The game continuously updates the snake's movement and features a dynamic scoring system, providing an interactive and engaging user experience. With keyboard controls for direction, this project exemplifies fundamental game development principles in Python, offering a simple yet enjoyable gaming encounter through graphical elements and user interactions.

.Art Generator project.

The Art Generator Python project is a thrilling exploration of merging computer vision and art. It uses the OpenCV library to turn regular images into intriguing ASCII art. The code takes an image file, transforms it to grayscale, and resizes it for processing. The unique aspect lies in mapping different pixel intensities to a set of chosen ASCII symbols and thresholds. Users can customise symbols and intensities, resulting in a visually stunning representation where the original image's details are translated into a mosaic of characters, offering a creative perspective on digital art.

.BMI project.

The BMI Python project functions as a straightforward Body Mass Index (BMI) calculator. Users input their weight in kilograms and height in centimeters, and the program computes the BMI by dividing weight by the square of height (converted to meters). The resulting BMI is displayed, and users receive feedback on their weight status, categorised as "severely underweight," "underweight," "healthy," "overweight," or "severely overweight." This project aims to provide a quick assessment of BMI and encourage users to consider their health in relation to established standards.

.Calculator project.

The Calculator Python project, built with PyQt5, creates a sleek and responsive graphical calculator. With an elegant design and user-friendly interface, it supports basic arithmetic operations such as addition, subtraction, multiplication, and division. Users can input numbers and decimals, and the "Evaluate" button promptly computes and displays the result. The "Back" button allows for correction by removing the last character, while the "AC" button provides a quick reset. This project showcases the seamless integration of Python and PyQt5, delivering an efficient and visually appealing calculator for straightforward calculations.

.Voice Recognition project.

The Voice Recognition Python project uses the SpeechRecognition library to create a voice-activated virtual assistant reminiscent of Alexa. By capturing voice commands through a microphone, the program leverages Google's speech recognition service to process and respond to various user requests. It can play songs on YouTube using pywhatkit, tell the current time, retrieve information from Wikipedia, and share jokes via pyjokes. This hands-free and interactive assistant showcases the integration of voice recognition technology in Python, offering a range of functionalities from entertainment to informative responses based on web searches, ensuring continuous user engagement.
